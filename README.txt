Comment exécuter notre projet ?
- Extraire l'archive "Code source"
- Ouvrir le projet Chess dans un IDE, Eclipse de préférence
- Exécuter dans un premier temps la classe "Serveur" dans le package uqac.aop.network
- Exécuter plusieurs instances de la classe "Client" dans le même package
- Tester l'application

Il est également possible de tester l'application à partir d'une console windows
- Exécuter la classe "Serveur"
- Ouvrir un terminal windows
- Exécuter la commande telnet localhost 1234
- Tester l'application
